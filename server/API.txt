*Описание API

API твитов, три метода:
* POST /users/[user_id]/tweets — добавляет твит от имени юзера user_id. Текст твита передаётся в параметре tweet, приклеенный файл — в параметре file.
*  GET /users/[user_id]/tweets — отдаёт список твитов юзера user_id. Также на странице будут имя и фамилия юзера.
*  GET /users — отдаёт список всех юзеров со ссылками на их страницы твитов.

API авторизации, два метода:
*  GET /auth/ редиректит юзера на страницу авторизации яндекса.
*  GET /auth/yandex/callback/ — коллбек авторизации, в котором мы получаем данные пользователя.
